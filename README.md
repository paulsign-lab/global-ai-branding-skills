# 글로벌 AI 브랜딩 — Claude Code 스킬 패키지

「AI 시대에 살아남는 디지털 브랜딩 실무」 강의의 리서치 → 전략 → 콘텐츠 전 과정을
**Claude Code 스킬 + MCP**로 묶은 패키지입니다. 강의에서는 각 작업을 나눠서
보여드리지만, 이 패키지를 쓰면 **한 대화창에서 통합**으로 돌릴 수 있습니다.

> 워크플로우를 직접 편집(Opal)하셔도 좋고, 이 스킬로 편하게 결과물을 받아보셔도 됩니다.
> **AI가 반복 작업을 처리하고, 사람은 본질적 판단과 전략 설계에 집중합니다.**

**저장소:** https://github.com/paulsign-lab/global-ai-branding-skills

## 빠른 시작
1. **설치 매뉴얼**을 따라 하세요 → [`INSTALL.md`](INSTALL.md)
2. MCP 연결: `.mcp.json` 템플릿 사용 (Semrush + Apify Reddit)
3. 스킬 흐름과 사용법 → [`skills/README.md`](skills/README.md)

## 들어있는 것
```
global-ai-branding-skills/
├─ INSTALL.md           설치 매뉴얼 (Antigravity + Claude Code + MCP)
├─ mcp.json.template    MCP 연결 템플릿 → .mcp.json 으로 이름 변경해 사용
├─ .gitignore
└─ skills/              스킬 8종 + 흐름 안내(README)
   ├─ competitor-quant/   (Part 3) Semrush 정량 분석
   ├─ reddit-voc/         (Part 3) 레딧 정성 분석 + 페르소나/FGI 질문지
   ├─ persona-fgi/        (Part 3) 가상 FGI 설계
   ├─ brand-mission/      (Part 4) 미션·비전
   ├─ brand-positioning/  (Part 4) 포지셔닝·2x2 맵
   ├─ brand-story/        (Part 4) 스토리·메시지 체계
   ├─ brand-voice/        (Part 4) 일관 콘텐츠 엔진
   └─ seo-content/        (Part 5) SEO·GEO 콘텐츠 자동 생성
```

## 필요한 계정 / 도구
- **Semrush** (Pro 이상) — MCP 기본 포함
- **Apify** (무료, 월 $5 크레딧) — 레딧 수집
- **Node.js**, (일부) **uv** — 설치 매뉴얼 참조
