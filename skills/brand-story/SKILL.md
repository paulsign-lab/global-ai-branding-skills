# brand-story — 브랜드 스토리 & 메시지 체계

미션·포지셔닝이 '설계도'라면, 스토리텔링은 거기에 **생명을 불어넣는 과정**이다.
이야기는 단순 팩트보다 기억에 오래 남고, 가격경쟁을 벗어나게 하며, 문화를 초월한다.
이 스킬은 확정된 전략을 [갈등→발견→결심→약속] 서사로 풀고, 실무에서 바로 쓰는
메시지 체계로 변환한 뒤, 진정성까지 검증한다.

## 언제 사용하나
- 미션·비전·포지셔닝 확정 후 브랜드의 '이야기'를 만들 때
- About 페이지·태그라인·히어로카피·SNS 훅 등 실무 카피가 필요할 때

## 입력값 (★ 앞 스킬 산출물 연결)
- **확정된 Mission / Vision / Positioning** ← `brand-mission`, `brand-positioning`
- **경쟁사 페인포인트 #1(가장 큰 기회)** ← `competitor-quant`
- **FGI에서 가장 감정이 강했던 발언 3개** ← `persona-fgi`
- **소비자 원래 언어(VoC) 핵심 표현 5개** ← `reddit-voc`
- 톤앤매너 키워드 / 창업자·브랜드 실제 배경(있으면)

## 실행 절차

### Part A — 오리진 스토리 3버전
서사 구조 필수: **[갈등]**(경쟁사 페인포인트 #1에서 도출, 구체적 상황) →
**[발견]**(FGI 발언의 좌절 반영) → **[결심]**(확정 미션과 연결) → **[약속]**(확정 비전과 연결)
- 버전 A 창업자 중심 / 버전 B 소비자 중심(FGI 실제 언어 반영) / 버전 C 산업문제 중심
- 각 250~300자 + 한 줄 요약(50자) + 적합 상황

### Part B — 메시지 체계 5단계 (위→아래로 짧고 행동유도적)
1. **브랜드 매니페스토** (8~10문장, About/브랜드북)
2. **태그라인** (10단어 이내, 후보 3개+추천, 영문+한글, 페인포인트 우회 공략)
3. **웹사이트 히어로 카피** (헤드라인=페인포인트 직접 공략+소비자 언어 / 서브헤드 / CTA)
4. **SNS 바이오** (160자, 후보 3개)
5. **SNS 콘텐츠 훅 5개** (릴스/숏폼 첫 3초, 페인포인트+소비자 언어 기반)

### Part C — 검증 체크리스트
- ① 진정성: 경쟁사 이름 넣어도 성립하는 문장 지적+수정
- ② 일관성: 미션→포지셔닝→스토리→매니페스토→태그라인이 한 흐름인가, 관통 키워드는?
- ③ 페르소나 공감 시뮬레이션: 대표 페르소나 2명 시점의 첫인상·공감·시도의향
- ④ 글로벌 적합성: 타깃 시장 문화 오해 표현? "Made in [원산지]" 시너지?
- → 수정 필요 부분을 수정 전/후로 제시

## 산출물
- 오리진 스토리 3버전 (+요약·적합상황)
- 메시지 체계 5종 (매니페스토~SNS 훅)
- 검증 4종 결과 (수정 전/후 포함)

## 🔗 핸드오프
- ← `brand-mission`, `brand-positioning`, `competitor-quant`, `persona-fgi`, `reddit-voc`
- → **`brand-gem`**: 매니페스토·톤앤매너·태그라인이 콘텐츠 생성 엔진의 '브랜드 규칙'으로.
- → **`seo-content`**: 메시지·소비자 언어가 SEO 콘텐츠 톤·헤드라인에 반영.

## 작성 원칙
- 스토리는 "지어내는 것"이 아니라 "진짜를 발견하고 다듬는 것"이다(Theranos 교훈).
  거짓 위에 세우지 않는다 — AI 생성 스토리도 사실 검증 필수.
- 소비자 원어는 의역하지 말고 그대로 녹인다.
