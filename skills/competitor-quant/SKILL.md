# competitor-quant — Semrush 기반 경쟁사 정량 분석

브랜딩 리서치의 출발점. **"남들이 어디에 깃발을 꽂았는지 알아야 우리가 꽂을 자리가
보인다."** 이 스킬은 Semrush MCP로 경쟁사의 검색 자산을 정량적으로 해부해, 우리가
점령할 **빈 곳(기회 키워드)**과 소비자가 검색으로 표출한 **페인포인트 키워드**를
숫자 근거와 함께 도출한다.

## 언제 사용하나
- 경쟁사 분석으로 우리 브랜드가 설 '빈 자리'를 찾을 때
- 미션·포지셔닝에 넣을 **검색량 기반 페인포인트 TOP5**가 필요할 때
- 레딧 정성 분석(`reddit-voc`)과 교차 검증할 정량 데이터를 만들 때

## 사전 준비
- **Semrush MCP 연결** 필요 (Pro 플랜이면 충분, 월 50,000 API 유닛 포함). `/mcp`로 확인.

## 입력값 (없으면 사용자에게 물어본다)
- **우리 브랜드 도메인** (또는 벤치마킹 대상)
- **경쟁사 도메인 2~3개** (예: cosrx.com, beautyofjoseon.com, anua 등)
- **타깃 시장/국가** (예: 미국 — 검색 데이터의 국가 설정)
- **카테고리 핵심 키워드** (예: korean sunscreen, snail mucin)

## 실행 절차

### STEP 1 — 경쟁사 오가닉 키워드 확보
Semrush `organic_research`/`domain` 도구로 각 경쟁사의 상위 오가닉 키워드를
검색량·순위와 함께 수집한다. 경쟁사별 상위 20~30개.

### STEP 2 — 키워드 갭 분석 (경쟁사 약점 탈취)
Semrush 키워드 갭으로 **경쟁사는 순위가 있는데 우리는 없는 키워드**를 찾는다.
= 새로 점령할 기회 키워드. 검색량·키워드 난이도(KD)와 함께 정리한다.

### STEP 3 — 페인포인트 키워드 추출
브랜드명/카테고리와 함께 검색되는 **부정·문제 의도 키워드**를 발굴한다.
(refund, review, issue, alternative, "끈적임", "fungal acne" 등)
질문형(Questions) 키워드도 함께 뽑아 소비자 정보 격차를 파악한다.

### STEP 4 — 정량 종합
- **페인포인트 키워드 TOP 5** (검색량 순) — 다음 파트 미션/포지셔닝 입력값
- **기회 키워드 TOP 10** (갭 + 적정 KD) — 선점 대상
- 경쟁사별 트래픽·키워드 규모 비교표

## 산출물
1. 경쟁사 오가닉 키워드 표 (키워드·검색량·순위·KD)
2. **기회 키워드 리스트** (키워드 갭, 우리가 점령할 자리)
3. **페인포인트 키워드 TOP 5 + 검색량** ← 핵심 핸드오프 데이터
4. 경쟁사 정량 비교 요약
> 마크다운 요약 + Excel로 정리한다.

## 🔗 핸드오프 (이 스킬의 출력이 어디로 가나)
- → **`reddit-voc`**: STEP 3 페인포인트 키워드를 레딧 정성 분석과 **교차 검증**
  ("검증된/잠재/검색수요만" 3분류)에 사용.
- → **`brand-mission`**: STEP 4의 "페인포인트 키워드 TOP5 + 검색량"이 미션 설계
  프롬프트의 `■ 경쟁사 페인포인트 TOP 5 (검색량)` 입력값으로 그대로 들어간다.
- → **`brand-positioning`**: 경쟁사 현재 포지셔닝·페인포인트가 2x2 맵 작성에 사용.
- → **`seo-content`**: 기회 키워드가 SEO 콘텐츠의 타깃 키워드로 연결.

## 작성 원칙
- 모든 주장에 **검색량·KD 숫자**를 붙인다(정량의 강점은 '증거').
- 국가 설정을 명시한다(미국 데이터인지 한국 데이터인지).
- 정량은 '무엇이 큰가'를 보여줄 뿐, '왜'는 `reddit-voc`·`persona-fgi`가 보완한다.
